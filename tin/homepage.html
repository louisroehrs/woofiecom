Make ONElist your Homepage

Follow the instructions below for your browser to make ONElist your start page.

Netscape 3.0 and below
Netscape 4.0 and above
Internet Explorer 3.0 and below
Internet Explorer 4.0 and above

Netscape 3.0 and below

To make ONElist your start page:

  1. From the Netscape pull-down Options menu, select General Preferences.
  2. Select the Appearance tab.
  3. In the Startup section of the page, click the Home Page Location radio button and enter http://www.onelist.com/ in the text entry box below the button.
  4. Click the OK button.

The next time you start Netscape or click the Home button you'll go directly to ONElist.

Netscape 4.0 and above

To make ONElist your start page:

  1. From the Netscape pull-down Edit menu, select Preferences.
  2. Select the Navigator from the Category list.
  3. In the Navigator Starts with section, click the Home Page radio button, and enter http://www.onelist.com/ in the Location text entry box in the Home Page section.
  4. Click the OK button.

The next time you start Netscape or click the Home button you'll go directly to ONElist.

Internet Explorer 3.0 and below

To make ONElist your start page:

  1. Go to the address http://www.onelist.com/
  2. From the Internet Explorer pull-down View menu, select Options.
  3. Click the Navigation tab.
  4. Select Start Page from the Page menu.
  5. Select the Use Current button and then click the OK button.
The next time you start Internet Explorer or click the Home button you'll go directly to ONElist.

Internet Explorer 4.0 and above

To make ONElist your start page:

  1. Go to the address http://www.onelist.com/
  2. From the Internet Explorer pull-down View menu, select Internet Options.
  3. Select the General tab.
  4. In the Home Page section, select the Use Current button and then the OK button.
The next time you start Internet Explorer or click the Home button you'll go directly to ONElist.

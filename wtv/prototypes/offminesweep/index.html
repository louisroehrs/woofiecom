How to Play
  		• The object of the game is to uncover all the squares in the minesweeper grid which don't contain mines.
  		• Uncover squares by clicking on them.
  		• Squares contain either a number (0-8) or a mine.
  		• If you click on a square with a mine below it the entire playing field is revealed and the game is over.
  		• If you reveal a number the game continues.

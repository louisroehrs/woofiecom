**KaraokeTV** — `index.html` is ready to open in a browser.

## What it looks like

- **Retro wood-grain TV body** with rounded bezel, chrome details, and CRT scanline + vignette effects
- **Two antennas** on top, two legs on the bottom
- **Green phosphor CRT screen** with animated static when idle
- **Three control areas** at the bottom: LED indicator + file/playback buttons on the left, an orange channel display in the middle, and a draggable volume knob on the right

## How to use it

1. **Open `index.html`** in a modern browser
2. **Expand "Advanced Setup"** and either:
   - Paste a public SoundFont URL (SF2/SF3) into the SoundFont field, **or**
   - Click "Load SF2/SF3 from disk" to pick one from your computer
   - (The SpessaSynth processor URL is pre-filled from jsDelivr)
3. **Drop a `.kar` or `.emk` file** onto the screen, or click the 📂 button
4. The song auto-plays — lyrics appear with green phosphor glow

## Controls

| Action | How |
|--------|-----|
| Play / Pause | ▶ button or `Space` |
| Stop | ■ button or `S` |
| Open file | 📂 button or `O` |
| Volume | Drag the knob up/down |
| Seek | Click the progress bar |
| Drag & drop | Drop a `.kar` anywhere on the screen |

> **SoundFont note:** The library requires a SoundFont file for MIDI synthesis. A good free option is [GeneralUser GS](https://schristiancollins.com/generaluser.php) — download the SF2 and load it from disk via the Advanced Setup panel.

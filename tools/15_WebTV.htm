Chapter 15

WebTV Extensions

WebTV has created HTML extensions to enhance the appeal of Web pages as displayed on TV sets. Computer-based browsers (such as Netscape Navigator and Microsoft Internet Explorer) will simply ignore any WebTV extensions in the HTML code in your Web pages.

Assumptions - For the sake of clarity, we want to make sure that readers understand that:

· All HTML elements need to be enclosed within angle brackets (for example,
<A HREF>).

· Most of the HTML tags are containers, so they need to have beginning and end tags.

All the WebTV extensions currently supported by the WebTV interface are listed and briefly described in Table 15-1, "WebTV Extensions" below.

Table 15-1
Tag

 Attribute

 Description

<A> 1 Creates a link around a specified text or graphic.
SELECTED Used in conjunction with the HREF attribute, determines that an anchor will be selected with a yellow highlight box.
<AUDIOSCOPE> For multimedia Web pages, displays animated images of sounds; in other words, displays graphically the amplitude of the current sound over time.
ALIGN Aligns the audioscope with the preceding text.
BORDER Sets the width of the audioscope border in pixels.
GAIN Sets a multiplier for the amplitude display of the audioscope.
HEIGHT Sets the height of the audioscope in pixels.
LEFTCOLOR Sets the color of the line that displays the audioscope's left audio channel.
LEFTOFFSET Sets the vertical offset for the display of the audioscope's left audio channel.
MAXLEVEL Specifies whether the audioscope should clip the sound according to the specified gain.
RIGHTCOLOR Sets the color of the line that displays the audioscope's right audio channel.
RIGHTOFFSET Sets the vertical offset for the display of the audioscope's left audio channel.
WIDTH Sets the width of the audioscope in pixels.
<BLACKFACE> As a typography standard, renders text in a double-weight boldface font.
<BODY> 1 Specifies the beginning and end of the body of a Web page.
CREDITS Sets the URL of a Web page that will display when the viewer selects the Credits button on the Info panel.
INSTRUCTIONS Sets the URL of a Web page that will display when the viewer selects the Instructions button in the Operations panel.
LOGO Sets the URL of a logo image to use in place of a Web page's thumbnail image.
<FONT> 1 Changes the appearance of a specified block of text in terms of size and/or color.
EFFECT Sets the font effect to one of three possible values: EMBOSS, RELIEF, or SHADOW.
TRANSPARENCY Allows the background to show through the text.
<HR> 1 Creates a horizontal rule.
INVERTBORDER Draws a horizontal rule that seems raised from the surface of the page.
<IMG> 1 Inserts an image within text (also known as an "inline" image).
SELECTED · Used in conjunction with the SRC and ISMAP attributes, specifies a value for the x,y pair for the initial cursor position in the image map.

· Used in conjunction with the SRC and USEMAP attributes, specifies a value for the index of an area in the initial selection of the associated client-side map.

TRANSPARENCY Used in conjunction with the SRC attribute, allows the background to show through an image.
<INPUT> 1 Used inside a form, defines the type of input field.
BGCOLOR Used in conjunction with the TEXT value for the TYPE attribute, sets the background color for the text entry field.
CURSOR Used in conjunction with the TEXT value for the TYPE attribute, sets the color for the cursor in the text entry field.
USESTYLE Used in conjunction with any of the values for the TYPE attribute, renders text in the control with the style currently in effect for the page.
WIDTH Used in conjunction with any of the values for the TYPE attribute, sets the width of the control in pixels.
<MARQUEE> 2 Creates a scrolling text marquee.
TRANSPARENCY Sets the marquee's background transparency.
<SELECT> 1 Used inside a form, creates a list of items from which the viewer can choose one or more.
AUTOACTIVATE Activates the list automatically when the viewer selects it.
Can be used in conjunction with the MULTIPLE and SIZE attributes.
BGCOLOR Sets the background color for the list.
Can be used in conjunction with the MULTIPLE and SIZE attributes.
SELCOLOR Sets the background color for the list selections.
Can be used in conjunction with the MULTIPLE and SIZE attributes.
TEXT Sets the text color for list items.
Can be used in conjunction with the MULTIPLE and SIZE attributes.
USESTYLE Renders list text in the style currently in effect for the page.
<TABLE> 1 Creates a table.
GRADANGLE Defines the gradient angle of a table.
GRADCOLOR Defines the ending color of a background gradient in a table.
TRANSPARENCY Sets the table's background transparency.
<TD> 1 Creates an individual table cell.
GRADANGLE Defines the gradient angle of a table cell.
GRADCOLOR Defines the ending color of a background gradient in a table cell.
TRANSPARENCY Sets the cell's background transparency.
<TEXTAREA> 1 Used inside a form, defines a text input area.
BGCOLOR Sets the text area's background color.
CURSOR Sets the color for the cursor in the text area.
USESTYLE Renders text in the text area in the style currently in effect for the page.
<TH> 1 Specifies a table heading cell.
GRADANGLE Defines the gradient angle of a table heading cell.
GRADCOLOR Defines the ending color for a background gradient in a table heading cell.
TRANSPARENCY Sets the background transparency for a table heading cell.
<TR> 1 Specifies a row of table cells.
TRANSPARENCY Sets the background transparency for cells in a row.
Legend:

1 denotes that the HTML tag was developed by the W3C/CERN.

2 denotes that the HTML tag was developed by Microsoft Internet Explorer.

All other tags and all the attributes listed here have been developed and are currently supported by WebTV.
